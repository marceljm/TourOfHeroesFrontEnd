# TourOfHeroesFrontEnd
<p>It's an Angular 2 project based on "Tour of Heroes" tutorial for TypeScript.</p>
<b>Installing, Compiling and Running...</b>
<p><i>1) cd TourOfHeroesFrontEnd/angular2-tour-of-heroes</i></p>
<p><i>2) npm install</i></p>
<p><i>3) npm start</i></p>

<p>Access http://localhost:3000/</p>
